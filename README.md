# Usage:
```
ftp_recurse -u FTP_USER -p FTP_PASS -h FTP_HOST

-s, --simulate	Pretend to mirror the remote directory but do not retrieve files
		or create directories.  Useful for confirming you've got your
		paths set up right

-s_pref SOURCE	Specify a directory to mirror relative to the home directory of
		of the FTP user

-t_pref TARGET	Specify a directory where remote content should be replicated
```
